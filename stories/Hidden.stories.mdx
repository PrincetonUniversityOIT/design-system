# Hidden Content

Sometimes it is necessary to hide content visually, but allow it to remain accessible to screen readers.

## Usage Notes

Use the ```puds-visually-hidden``` or ```puds-sr-only``` class on any HTML element that you want to hide.

## Implementation Notes

To accomplish this, width, height, margin, padding, borders, and word wrap are all removed to attempt to remove 
all visual impact of the component, while leaving the content in the document.

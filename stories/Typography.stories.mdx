# Typography

Web Fonts are used for typography in this Design System.

Take advantage of the fact that font files do not change often by specifying appropriate HTTP headers for the font
files (Cache-Control, ETag, and Last-Modified).

Font display behavior (CSS font-display attribute) is controlled via the font-display-preference SCSS variable
in the _variables.scss file.  The recommended setting is "swap" to prefer an extremely small block period (faster
rendering) and an infinite swap period (no limit on how long it takes to swap in the web fonts once they are loaded).

The ```font-size``` has been set to ```100%``` on the ```html``` element.  This has been done to void setting it to 
a static value that the user would be unable to modify using browser font size preferences.  Since all ```rem```
units will be based off of the ```font-size``` value, this is an important design decision. A value of 100% is the 
equivalent of 16px in all browsers, by default, but the user can modify this value through browser configuration. 

Because of a bug in WebKit/Safari (https://bugs.webkit.org/show_bug.cgi?id=156684), media queries should use
```em``` units instead of ```rem``` units.  Both ```em``` and ```rem``` units are based on the initial ```font-size```,
which in most browsers is 16px.

## Headings

When picking a heading, document structure should be considered before the style.  Heading tags should be chosen
to create properly nested headings that never skip levels.

After document structure has been properly honored, choose the appropriate heading style to style the heading
to the size and style that meets the needs of your page.

The heading stories that follow demonstrate the ability to style any heading tag (h1, h2, h3, etc...) to any heading
style.

## Vertical Rhythm

The "vertical rhythm" of the page is mostly tied to ```line-height``` because text on the page is spaced and flows mostly 
based on the ```line-height``` value.

## Implementation Notes

* Setting the ```font-size``` CSS descriptor on the html element to anything other than 100% (or 1rem) will disable
  the ability for people to set custom font-sizes in their browser.

Typography

The Design System typography allows developers to easily apply a default set of font weights, font sizes, lineheights, and letterspacings.

Headings

<h1 class="jazz-l">Heading One As Heading L</h1>
<h2 class="jazz-l">Heading Two As Heading L</h2>
<h3 class="jazz-l">Heading Three As Heading L</h3>
<h4 class="jazz-l">Heading Four As Heading L</h4>
<h5 class="jazz-l">Heading Five As Heading L</h5>
<h6 class="jazz-l">Heading Six As Heading L</h6>

Heading styles are pre-defined as T-Shirt sizes—jazz-xxl, jazz-xl, jazz-l, jazz-m, jazz-s, jazz-xs. Adding a T-shirt size to a heading's class will apply the pre-defined style as specified in the table below:

Using the pre-defined heading style will override its default styles, therefore using any one of the existing HTML headings—h1, h2, h3, h4, h5, h6—with a T-shirt size will result in a uniform appearance. While multiples of the same HTML heading can be used with different T-shirt sizes to simulate multiple heading styles, this practice is not recommended for reasons pertaining to search engine crawlers and SEO best practices. It is important to keep this in mind and use different heading tags in relation to its priority.

Body

<p class="jazz-body-reg">Paragraph as Body Regular</p>
<p class="jazz-body-sm">Paragraph as Body Small</p>
<p class="jazz-body-sm-m">Paragraph as Body Small Medium</p>
<p class="jazz-body-semi">Paragraph as Body Semi Bold</p>
<p class="jazz-body-bold">Paragraph as Body Bold</p>

Body styles are pre-defined in 5 sizes—jazz-body-reg, jazz-body-sm, jazz-body-sm-m, jazz-body-semi, jazz-body-bold. Adding a body style to an HTML tag will apply the pre-defined style as specified in the table below:

Any HTML element without a specified style will automatically default to use Body Regular styles. Else, using the pre-defined body style will override an HTML tag's default styles.
